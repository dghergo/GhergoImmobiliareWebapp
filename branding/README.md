# Branding Assets - Ghergo Immobiliare

Questa cartella contiene le risorse di branding del cliente.

## Contenuto

### Logo
Inserire qui i file del logo in diversi formati:
- Logo principale (preferibilmente SVG o PNG ad alta risoluzione)
- Varianti (orizzontale, verticale, icona)
- Versioni chiare/scure se disponibili

**Esempio di naming:**
- `logo-ghergo.svg`
- `logo-ghergo-orizzontale.png`
- `logo-ghergo-icon.png`

### Colori
Inserire qui la palette colori del brand:
- File di specifica colori (PDF, AI, Figma export, etc.)
- Screenshot della palette
- Codici HEX/RGB/HSL

**File consigliati:**
- `palette-colori.pdf`
- `colori.txt`
- `brand-guidelines.pdf`

### Note
- Mantenere i file originali ad alta qualità
- Documentare eventuali varianti di colore
- Includere informazioni su font se disponibili
